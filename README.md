# Love & Happiness

A charming one-page web experience built with HTML, CSS, and vanilla JavaScript. The page features a cozy cat-themed card, a playful "YES or NO" interaction, and a soft celebration popup when the user chooses "YES".

## Features

- A warm, romantic one-page layout with a sweet cat illustration
- A "NO" button that dodges the cursor by moving to random positions inside the card
- A joyful popup that appears when the user clicks "YES"
- Fully loaded and rendered dynamically through JavaScript

## Project Structure

- [index.html](index.html) — basic page shell
- [styles.css](styles.css) — visual styling and animations
- [script.js](script.js) — dynamic page generation and interactions
- [assets/cat.svg](assets/cat.svg) — local cat illustration

## Run Locally

From the project folder, start a simple local server:

```bash
python -m http.server 8000
```

Then open your browser at:

```text
http://localhost:8000
```

## Notes

The interaction is designed to feel lighthearted and playful while keeping the experience polished and easy to use.
